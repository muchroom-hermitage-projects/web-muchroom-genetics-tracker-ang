About

Mycology Genetics Tracker

This app tracks mushroom culture lineage, relationships, and key metadata across spores, agar, liquid culture, grain spawn, fruit, clones, and slants.

Current Data Format (version 1)

Data is persisted and imported/exported as JSON with this top-level shape: 

{{ dataFormatExample }}

Culture dates are serialized as JSON date strings and restored to Date objects on import.
